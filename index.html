Manage Profiles

How to Customize

You can add your own 3D print designer usernames, profile URLs, or links to specific models by editing the local profiles.config.json file in your directory.

File Structure: profiles.config.json 
{
  "profiles": [
    {
      "platform": "printables",
      "username": "Prusa3D"
    }
  ],
  "models": [
    "https://www.printables.com/model/17215"
  ]
}

Running the Sync Utility

To update the database (models.json) with your configured profiles and links:

node fetch_models.js
